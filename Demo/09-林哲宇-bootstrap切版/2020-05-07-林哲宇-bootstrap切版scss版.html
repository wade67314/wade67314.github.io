石川祐希（いしかわ ゆうき）

石川祐希

1995年出生，今年年僅24歲的「東亞最強排球傳奇」球員「石川祐希」，高中時就以王牌之姿活躍於排球界，將當時並不是名校的星城高中拉至新高度，帶領母校奪得全國賽六連霸！ Go somewhere

東亞最強排球傳奇

石川說是日本頂級的主攻手絕對沒有人會有二話，連世界排球協會都將他列入「如果可以組一支排球夢幻隊伍的話想納入的球員」名單，日本則稱他為「平成年代最後希望」，世界排球賽中他也因為優異表現獲得「最佳主攻手」殊榮，可見他在日本人心中的地位！ Go somewhere

同樣也是日本世界級球員「西田有志」只有19歲，是2000年出生，被稱為日本排壇「怪物新人」，將與前輩石川祐希一同征戰2020東京奧運，很期待這兩位「東亞傳奇」的球員到時會如何發揮「彷彿漫畫走出來的超強實力」！

被日本譽為平成年代最後希望的石川祐希，不僅寫下日本最年輕旅義球員的紀錄，更在球季後半展現出他強大的得分能力，屢屢為球隊拿下最多得分紀錄的他，儼然成為Siena的關鍵球員 (Key player)，得分紀錄更在Serie A1裡排上第12位。究竟，他賽季的中後期他做了什麼樣的改變，讓他能夠有這樣突飛猛進的發展？

談到球季後半得分率上升的原因，石川不吝嗇地回答：「聯盟裡的球員都非常高，想要突破他們的攔網是一件不太容易的事情，因此我嘗試把揮臂做得更快，讓球從他們做完攔網之前就已經打掉了。」而幫助石川能快速揮臂的正是他將右手肘儘可能地向後伸展！石川更提到，右手肘向後延伸的拉弓動作不單只是拉弓而已，更重要的是可以帶動腰部的旋轉。透過上肢的拉弓來引導腰部向右旋轉，在決定要攻擊揮臂的剎那再將腰部向左旋轉，帶動上肢能夠更快的揮臂。
